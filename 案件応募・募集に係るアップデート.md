# 薬局求人システム：完全仕様書

## 重要な前提

### 報酬・手数料の支払いタイミング
1. **プラットフォーム手数料**（薬局 → プラットフォーム）
   - 支払い期限：初回出勤日の3日前まで
   - 支払い確認後に個人情報を開示

2. **薬剤師への報酬**（薬局 → 薬剤師）
   - 支払い期限：体験期間終了後
   - プラットフォームは関与しない（当事者間で直接支払い）
   - 理由：先払いだと持ち逃げして飛ぶ可能性があるため

### 募集成功のポイント
📌 **長期間（60〜90日）または短期間で高単価の求人ほど応募が集まりやすい傾向があります**
- 長期間：安定した収入を求める薬剤師にアピール
- 短期高単価：効率的に稼ぎたい薬剤師にアピール
- 中途半端な日数（20〜40日程度）は応募が集まりにくい可能性があります

### フェーズ1：募集フェーズ

#### 【ステップ1】求人投稿

**薬局側の入力項目：**

##### 必須項目
1. **求人タイトル***
   - 入力形式：テキスト入力

2. **求人詳細**
   - 入力形式：テキストエリア
   - 内容：仕事内容、職場の雰囲気など

3. **勤務地**
   - 入力形式：テキスト入力
   - 例：大阪市中央区

4. **希望勤務日数***
   - 入力形式：数値（10〜90日）
   - 補足テキスト：「10日〜90日（3ヶ月）の範囲で入力してください」
   - リアルタイム計算表示：「報酬総額：◯◯万円（日給2.5万円 × ◯日）」
   - 💡ヒント表示：「長期間（60〜90日）または短期間で高単価の求人は応募が集まりやすい傾向があります」

5. **勤務開始可能期間***
   - 入力形式：日付選択
   - 制限：今日から2週間後以降の日付のみ選択可能
   - ラベル：「この日から2週間の間で初回勤務日を相談」
   - 補足テキスト：「薬剤師と相談の上、初回勤務日を決定します」

6. **募集期限***
   - 入力形式：日付選択
   - デフォルト値：今日から7日後
   - 入力制限：最短3日後、最長14日後

7. **応募条件・資格**
   - 入力形式：テキストエリア
   - 例：薬剤師免許必須、調剤経験3年以上

##### 任意項目（参考情報）
8. **希望勤務時間帯**
   - 入力形式：テキストエリア
   - プレースホルダー：「例：平日9:00-18:00希望」
   - 補足テキスト：「※あくまで希望です。実際の勤務時間は薬剤師と相談の上決定します」
   - 注記：具体的な勤務曜日・スケジュールは双方の合意のもと後から決定

```
求人が公開され、薬剤師が検索・閲覧可能な状態になる
```

```
   └─ 薬剤師が候補から1つ選択して回答
```

**■ 条件確認項目（都度確認可能、承認/否認フローなし）**
```

├─ 報酬総額：◯◯万円（日給2.5万円 × ◯日）
│  └─ ※薬局から薬剤師への支払いは体験期間終了後

   └─ ※具体的な勤務曜日・スケジュールは双方の合意のもと後から決定

※薬局・薬剤師双方が何度でも確認可能
```

#### 【ステップ6】正式な採用オファー

**薬局側が「正式オファー」を送信**

```
├─ 報酬総額：◯◯万円（日給2.5万円 × ◯日）
│  └─ ※体験期間終了後に薬局から薬剤師へ直接支払い
├─ 勤務時間：◯◯時〜◯◯時（目安）
├─ プラットフォーム手数料：◯◯円
└─ 手数料支払い期限：初回出勤日の3日前まで
```

**重要事項の通知：**

### フェーズ5：未払い時の対応

#### 【ステップ12】未払いの場合の処理

**■ 初回出勤日の前日23:59時点で手数料未払いの場合：**
```
├─ 契約を自動キャンセル
├─ 薬局側にペナルティ
│  ├─ アカウント停止（新規募集不可）
│  ├─ 既存の募集も一時停止
│  └─ 通知：「手数料の支払い遅延により募集機能が停止されました」
├─ 薬剤師側に通知
│  └─ 「契約がキャンセルされました。ご迷惑をおかけして申し訳ございません」
└─ システム管理者に通知
   └─ 薬局の与信状況を確認
```

**■ ペナルティ解除の条件：**
```
├─ キャンセルされた契約分の手数料支払い完了
└─ システム管理者による審査・承認
```

**■ 再発防止：**
```
└─ 2回目の未払いでアカウント永久停止
```

```
契約成立
  ↓
プラットフォーム手数料請求書発行（薬局宛）
  ↓
薬局がプラットフォームに手数料を支払い
期限：初回出勤日の3日前まで
  ↓
手数料支払い確認
  ↓
個人情報開示（双方）
  ↓
初回出勤日
  ↓
体験期間（10日〜90日）
  ↓
体験期間終了
  ↓
薬局が薬剤師に報酬を直接支払い
金額：日給2.5万円 × 実際の勤務日数
  ↓
システム上での管理終了
```

## 主要な仕様まとめ

### 支払いの流れ

**プラットフォーム手数料（薬局 → プラットフォーム）**
- **タイミング**：契約成立後、初回出勤日の3日前まで
- **金額**：◯◯円（※手数料率は別途設定）
- **効果**：支払い確認後に個人情報を開示
- **未払い時**：契約自動キャンセル + ペナルティ

**薬剤師への報酬（薬局 → 薬剤師）**
- **タイミング**：体験期間終了後
- **金額**：日給2.5万円 × 実際の勤務日数
- **方法**：当事者間で直接支払い（システム不関与）
- **理由**：先払いだと持ち逃げリスクがあるため後払い

### 勤務開始日の制限
- 今日から2週間後以降の日付のみ選択可能
- 理由：手数料支払い期限（初回出勤日の3日前）を確保するため

### ペナルティ制度
- 手数料未払いで契約キャンセルになった薬局は募集機能停止
- 手数料支払い完了 + 管理者承認で解除
- 2回目の未払いで永久停止

### 勤務曜日・スケジュール
- システム上での設定・管理は不要
- 契約成立後、薬局と薬剤師が直接相談して決定

### 募集成功のポイント
- 長期間（60〜90日）または短期間で高単価の求人が集まりやすい
- 求人投稿画面にヒントとして表示

## 補足情報

### システムの設計思想
1. **薬剤師の安全性を最優先**：手数料支払い確認後に個人情報開示
2. **薬局の信頼性担保**：未払いペナルティで悪質な行動を抑制
3. **シンプルな設計**：体験期間中の管理は当事者に委ね、システムは最小限
4. **透明性の確保**：報酬・手数料・支払いタイミングを明確に表示

### 想定される利用シーン
- 産休・育休代替の短期募集
- 繁忙期の増員（インフルエンザシーズンなど）
- 新規開局時の試用期間採用
- 地方薬局の人材確保

### 競合優位性
- 日給固定（2.5万円）でシンプル
- 体験期間での相性確認が可能
- プラットフォーム手数料の先払いで薬剤師保護
- 長期・短期両方に対応した柔軟な日数設定
