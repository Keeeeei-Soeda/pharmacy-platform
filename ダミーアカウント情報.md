## 📝 アカウント作成方法

### 1. フロントエンド経由での新規登録

1. http://localhost:3000/auth/register にアクセス
2. 上記のメールアドレスとパスワードを入力
3. 適切なユーザータイプを選択
4. 登録ボタンをクリック

### 2. バックエンド API 経由での直接作成

```bash
# 薬剤師アカウント作成
curl -X POST http://localhost:3001/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "pharmacist@test.com",
    "password": "password123",
    "userType": "pharmacist",
    "firstName": "太郎",
    "lastName": "薬剤師"
  }'

# 薬局アカウント作成
curl -X POST http://localhost:3001/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "pharmacy@test.com",
    "password": "password123",
    "userType": "pharmacy",
    "firstName": "薬局",
    "lastName": "管理者"
  }'
```

## ⚠️ 注意事項

1. **バックエンドサーバーの起動が必要**

   - バックエンドが起動していない場合、ログインや API 呼び出しが失敗します
   - バックエンドサーバー: http://localhost:3001

2. **データベース接続**

   - PostgreSQL データベースが起動している必要があります
   - 環境変数（DATABASE_URL 等）が正しく設定されている必要があります

3. **初回登録時の自動プロフィール作成**
   - 薬剤師アカウント: 基本的な薬剤師プロフィールが自動作成されます
   - 薬局アカウント: 基本的な薬局プロフィールが自動作成されます
