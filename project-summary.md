# 薬剤師転職サイト LP制作 プロジェクトドキュメント

## 📋 プロジェクト概要

**プロジェクト名**: 薬剤師キャリアナビ ランディングページ制作  
**制作期間**: 2026年1月10日  
**目的**: トライアル勤務という革新的な仕組みを持つ薬剤師転職サービスのLPを制作

---

## 🎯 プロジェクトの背景

### 初期要件
最初の依頼では、**一般的な薬剤師転職サイト**としてのLP制作が求められました：
- 市場調査に基づいた競合分析
- TOP5薬剤師転職サイトのLP分析
- 薬剤師・薬局の2つの登録ボタン
- 一般的な転職サイトのメリット訴求

### 方針転換のきっかけ
プロジェクト資料（要件定義書・進捗レポート）の確認により、サービスの**真の強み**が判明：
- **トライアル勤務**という独自の仕組み
- 働いてから本採用を決定できる革新的なシステム
- 一般的な転職サイトとは全く異なる価値提案

この発見により、**差別化を明確にしたLP**へと方針を転換しました。

### Phase 4: コンテンツの全面刷新

#### メインメッセージの変更

**【変更前】一般的な転職サイト**
```
あなたの理想の
キャリアを実現。

50,000件以上の求人から、あなたに最適な職場を
専任アドバイザーと一緒に見つけましょう。
完全無料で転職をサポートします。
```

**【変更後】トライアル勤務を前面に**
```
働いてから決める、
新しい転職のカタチ。

トライアル勤務で職場を体験。
給料をもらいながら、本当に合う職場を見つけられる。
```

#### 統計データの変更

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| 1 | 50,000+ 掲載求人数 | **95%** トライアル後の本採用率 |
| 2 | 5,000+ 提携薬局・病院 | 5,000+ 提携薬局・病院 |
| 3 | 96.5% 転職成功率 | **100%** 満足保証（合わなければ次へ） |
| 4 | 3.14倍 有効求人倍率 | **0円** 薬剤師の利用料金 |

#### 薬剤師向けメリット（3つ→2つに厳選）

**1. 💼 実際に働いてから転職を判断**
- トライアル勤務で職場の雰囲気、人間関係、業務内容を体験してから本採用を決定できます。

**2. 💰 給料をもらいながら転職活動**
- トライアル期間中も時給が発生。収入を得ながら自分に合う職場を探せます。

**削除した項目**:
- 🔄 複数の職場を比較検討（重複感があるため統合）

#### 薬局向けメリット（3つ→2つに厳選）

**1. 👀 実際の働きを見てから採用**
- トライアル勤務で実際のスキル、人柄、職場との相性を確認してから本採用を判断できます。

**2. 💸 採用コストを大幅削減**
- 紹介手数料不要。トライアル期間の時給のみで、低コストで優秀な人材を採用できます。

**削除した項目**:
- 🛡️ 採用失敗リスクを大幅に軽減（1番目の項目に内包）

#### 新規セクションの追加

**「トライアル勤務とは？」セクション**

**トライアル期間中にできること**:
- ✓ 時給が発生（給料をもらいながら体験）
- ✓ 簡単な勤怠管理
- ✓ いつでも次の職場を探せる

**本採用前に確認できること**:
- ✓ 職場の雰囲気・人間関係
- ✓ 実際の業務内容・忙しさ
- ✓ 通勤のしやすさ
- ✓ 働きやすさ・相性

#### 削除したセクション
- ❌ 利用者の声セクション（実績データが不確定なため）
- ❌ LINE連携の記述（開発中の機能のため）

## 📝 制作メモ

### デザイン判断の記録

**なぜTeal + Blueを選んだのか**:
- Teal: 医療・ヘルスケア業界で信頼感を表現
- Blue: 誠実さ・安全性・安定性
- 組み合わせ: 革新的でありながら信頼できる印象

**なぜ写真を大きく使ったのか**:
- 実際の職場環境を視覚的に伝える
- 競合サイトとの差別化
- 感情的なつながりを作る

**なぜホバー時に青色に変わるのか**:
- 初期状態: 情報を読みやすく（白背景）
- ホバー時: アクション可能を示唆（青色）
- インタラクティブ性の強調

### 技術選定の理由

**Next.js 15を選んだ理由**:
- App Routerによる最新のルーティング
- サーバーコンポーネントの活用
- 優れたSEO対策
- 画像最適化の自動化

**Framer Motionを選んだ理由**:
- 宣言的なアニメーション記述
- React 19との相性
- パフォーマンスの良さ
- スクロール連動が簡単

**Tailwind CSS 4を選んだ理由**:
- 迅速なプロトタイピング
- デザインシステムとの相性
- レスポンシブ対応が簡単
- カスタマイズ性の高さ
