## ⚠️ 重要な確認事項

### 1. XServerのSSH設定画面で確認

#### A. SSH設定が有効になっているか
- [ ] 「SSH設定」が「有効(ON)」になっている
- [ ] 「無効(OFF)」の場合は「有効(ON)」に変更

- `yaku-navi.com` が `sv16713.xserver.jp` を指しているか確認
- または、`yaku-navi.com` が別のVPSサーバーを指している可能性

#### C. ユーザー名の確認
- XServerのレンタルサーバーの場合、ユーザー名は `xs621921`
- VPSサーバーの場合、ユーザー名は `pharmacy` の可能性

### 2. 接続テスト

以下のコマンドで接続を試してください：

```bash
# パターン1: XServerのレンタルサーバーに接続
ssh -i ~/.ssh/xs621921.key xs621921@sv16713.xserver.jp

# パターン2: yaku-navi.comにxs621921で接続
ssh -i ~/.ssh/xs621921.key xs621921@yaku-navi.com

# パターン3: yaku-navi.comにpharmacyで接続（VPSの場合）
ssh -i ~/.ssh/xs621921.key pharmacy@yaku-navi.com
```

### 3. XServerのSSH設定画面で表示されている接続例

XServerのSSH設定画面に、通常以下のような接続例が表示されています：

